# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.4] - 2026-04-11

### Added
- Python informational checks in `noxdev doctor` command for better environment validation
- Fullstack demo template with React + FastAPI scaffolding via `noxdev demo`
- "Works with" section in README with comprehensive fullstack demo screenshot
- Support for compound commands in `noxdev run` (e.g., `frontend && backend`)
- Type-specific configuration defaults in `noxdev init` with auto-detection from manifest files

### Enhanced
- Improved `noxdev demo` command to scaffold complete fullstack applications
- Enhanced project initialization with intelligent project type detection
- Updated documentation with better visual examples and compatibility matrix
